=====================
Ubuntu 源使用帮助
=====================

地址
====

https://mirrors.scau.edu.cn/ubuntu/

收录架构
========

AMD64 (x86_64), Intel x86

使用说明
========

编辑  /etc/apt/sources.list  文件

执行 ``sudo sed -i 's/cn.archive.ubuntu.com/mirrors.scau.edu.cn/g' /etc/apt/sources.list`` 

最后执行 ``sudo apt-get update`` 更新软件源列表
